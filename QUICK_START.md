# 🚀 Hızlı Başlangıç - 5 Adımda AdSense

## ⚡ Yapılması Gerekenler

### 1️⃣ Build Et (2 dakika)

```bash
npm run build
```

### 2️⃣ Hostinger'a Yükle (10 dakika)

- `dist` klasörünün içeriğini
- `public_html` klasörüne yükle
- FTP veya File Manager ile

### 3️⃣ Test Et (5 dakika)

```
✅ fahrieren.com
✅ fahrieren.com/robots.txt
✅ fahrieren.com/sitemap.xml
✅ fahrieren.com/ads.txt
✅ fahrieren.com/gizlilik-politikasi
✅ fahrieren.com/kullanim-kosullari
```

### 4️⃣ Google Search Console (5 dakika)

1. search.google.com/search-console
2. Site ekle → fahrieren.com
3. Doğrula
4. Sitemap gönder

### 5️⃣ AdSense'e Başvur (2 dakika)

1. adsense.google.com
2. Add site → fahrieren.com
3. Request review
4. 1-2 hafta bekle ☕

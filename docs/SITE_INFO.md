# Fahri Eren - Ticaret Platformu

## Hakkımızda

Fahri Eren olarak 1998 yılından bu yana Antalya, Yeşilova'da ticaret sektöründe faaliyet göstermekteyiz. Müşteri
memnuniyetini öncelik olarak belirleyerek, kaliteli ürün ve hizmet sunmayı ilke edindik.

## Web Sitesi Bilgileri

- **Domain**: fahrieren.com
- **Platform**: React + TypeScript + Vite
- **Dil**: Türkçe (TR) / İngilizce (EN)
- **Yayın Tarihi**: Ekim 2025
- **Google AdSense**: Aktif (ca-pub-2016267232144093)

## İçerik Yapısı

### Ana Sayfalar

1. **Ana Sayfa (/)**: Platform tanıtımı, öne çıkan ürünler, hizmetler
2. **Ürünler (/urunler)**: Tüm ürün kategorileri ve listesi
3. **Hakkımızda (/hakkimda)**: Şirket geçmişi, deneyim, referanslar
4. **İletişim (/iletisim)**: İletişim formu, harita, iletişim bilgileri
5. **İş Ortaklarımız (/is-ortaklarimiz)**: İş ortakları ve referanslar
6. **Gizlilik Politikası (/gizlilik-politikasi)**: KVKK uyumlu gizlilik politikası
7. **Kullanım Koşulları (/kullanim-kosullari)**: Web sitesi kullanım şartları

### Ürün Kategorileri

1. **Emlak (Real Estate)**
    - Konut satışı

    - Ticari gayrimenkul
    - Emlak danışmanlığı

2. **Araçlar (Vehicles)**
    - İkinci el araç alım-satımı
    - Araç aracılık hizmetleri

3. **Tarım Ürünleri (Agriculture)**
    - Yumurta üretimi (Eren Yumurta)
    - Tarım ürünleri satışı
    - Toptan ve perakende

4. **İnşaat Malzemeleri (Construction)**
    - İnşaat malzemesi tedariki
    - Toplu satış
    - Proje bazlı çalışmalar

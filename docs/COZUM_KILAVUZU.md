## 🚀 Şimdi Ne Yapmalısınız?

### Adım 1: Projeyi Build Edin

```bash
npm run build
```

Bu komut `dist` klasöründe production-ready dosyaları oluşturacak.

### Adım 2: Hostinger'a Yükleyin

**Seçenek A: FTP ile**

1. FTP istemcisi (FileZilla vb.) açın
2. Hostinger FTP bilgilerinizi girin
3. `dist` klasörünün içeriğini `public_html` klasörüne yükleyin

**Seçenek B: Hostinger File Manager**

1. Hostinger kontrol paneline giriş yapın
2. File Manager'ı açın
3. `public_html` klasörüne gidin
4. `dist` klasörünün içeriğini yükleyin

### Adım 3: Domain Kontrolü

Deploy sonrası bu URL'lerin çalıştığından emin olun:

```
✅ https://fahrieren.com/
✅ https://fahrieren.com/robots.txt
✅ https://fahrieren.com/sitemap.xml
✅ https://fahrieren.com/ads.txt
✅ https://fahrieren.com/gizlilik-politikasi
✅ https://fahrieren.com/kullanim-kosullari
```

### Adım 4: Google Search Console

1. https://search.google.com/search-console adresine gidin
2. "Add Property" ile sitenizi ekleyin
3. Domain ownership doğrulaması yapın
4. Sitemaps bölümünden sitemap.xml'i gönderin:
   ```
   https://fahrieren.com/sitemap.xml
   ```

### Adım 5: Google AdSense'e Tekrar Başvurun

1. https://adsense.google.com adresine gidin
2. Sites → "Add Site" tıklayın
3. fahrieren.com domain'ini girin
4. Verification code zaten `index.html`'de mevcut (ca-pub-2016267232144093)
5. "Request Review" butonuna tıklayın
6. 1-2 hafta içinde inceleme sonucunu bekleyin

## 🎓 İpuçları

### AdSense Review Süreci

- İnceleme süresi: **1-2 hafta**
- Sabırlı olun, erken başvuru yapmayın
- Red alırsanız: 2 hafta bekleyin, tekrar başvurun

### Red Alırsanız Ne Yapmalı?

1. Red nedenini dikkatlice okuyun
2. Belirtilen sorunları düzeltin
3. 2 hafta bekleyin (önemli!)
4. Tekrar başvurun

### Onay Alındıktan Sonra

1. Auto Ads özelliğini aktif edin (önerilir)
2. veya Manual placement ile reklamları kendiniz yerleştirin
3. Cookie consent mesajını aktif edin
4. Gelir raporlarını düzenli takip edin
